# CMO Roadmap

## Immediate Next Steps (Phase 1-2 Maintenance)
- [ ] **Email Notifications:** Plug in Resend or SendGrid to email the daily report results.
- [ ] **PDF Exporting:** Add a "Download Full PDF Report" button to the Chief Aggregator's output.
- [ ] **Subreddit Auto-Listening:** Expand Reddit agent search keywords beyond the initial product info extraction.

## Future Agents & Intelligence
- [ ] **Competitor Agent:** Enter a competitor's URL; CMO performs a side-by-side positioning audit.
- [ ] **Ad Spend Agent:** Estimate Google/FB ad spend required based on target keywords.
- [ ] **Technical Audit Agent:** Check Lighthouse/Speed insights alongside growth strategy.
- [ ] **Social Media Video/Visuals Agent:** Integrated hooks for TikTok/Reels/Shorts scripts.

## Platform Features
- [ ] **Team Collaboration:** Invite-only project workspaces for startups.
- [ ] **Zapier/Slack/Discord:** Push notifications to your team's workspace as soon as the daily analysis is ready.
- [ ] **AI Video/Image Assets:** Integration with Midjourney or Runway to generate ad creatives alongside copy.

## Scaling
- [ ] **Vercel KV/Postgres Integration:** Shifting from JSON-based file storage to a robust database for unlimited scaling.
- [ ] **Vector Search (RAG):** Indexing months of daily reports to see growth trends and pattern changes over time.
