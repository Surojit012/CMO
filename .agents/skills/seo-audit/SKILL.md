# SEO Audit Skill

## Frameworks

### 1. On-Page SEO Scorecard
Audit these elements for every page:
- **Title Tag**: 50-60 chars, keyword front-loaded, compelling click angle
- **Meta Description**: 150-160 chars, includes CTA, unique per page
- **H1 Tag**: Exactly ONE per page, contains primary keyword naturally
- **Header Hierarchy**: Logical H1 → H2 → H3 nesting (no skipped levels)
- **URL Structure**: Short, keyword-rich, no special characters
- **Image Alt Text**: Descriptive, keyword-relevant, not stuffed
- **Internal Links**: 3-5 relevant internal links per page
- **External Links**: Link to authoritative sources where relevant

### 2. Content-Market Fit SEO Framework
For each keyword target, evaluate:
1. **Search Intent Match**: Is the content type right? (informational / transactional / navigational)
2. **SERP Analysis**: What's ranking now? Can you create something 10x better?
3. **Keyword Difficulty vs. Domain Authority**: Can you realistically rank?
4. **Business Value**: Does ranking for this keyword drive revenue?
5. **Content Gap**: What do top 3 results miss that you can cover?

### 3. Keyword Research Process
1. **Seed Keywords**: Extract from product features, benefits, use cases
2. **Long-Tail Expansion**: "[seed] for [audience]", "best [seed]", "how to [seed]"
3. **Question Keywords**: "what is", "how to", "why does", "vs", "alternative to"
4. **Competitor Keywords**: What are competitors ranking for that you're not?
5. **Intent Buckets**: Sort into TOFU (awareness), MOFU (consideration), BOFU (decision)

### 4. Technical SEO Checklist
- [ ] Page loads in < 3 seconds
- [ ] Mobile-friendly (responsive design)
- [ ] HTTPS enabled
- [ ] XML sitemap submitted
- [ ] robots.txt properly configured
- [ ] No duplicate content / canonical tags set
- [ ] Schema markup (Organization, Product, FAQ, Article)
- [ ] Core Web Vitals passing (LCP, FID, CLS)
- [ ] No broken links (404s)
- [ ] Proper 301 redirects (no chains)

### 5. Content SEO Blueprint
For each piece of content, ensure:
- Primary keyword appears in: title, H1, first 100 words, last paragraph
- Secondary keywords woven naturally throughout
- Minimum 1,500 words for competitive topics (longer isn't always better)
- Table of contents for posts > 2,000 words
- FAQ section targeting "People Also Ask" queries
- Internal linking to 3+ relevant pages
- External linking to 2+ authoritative sources

### 6. Blog Post Ideas Formula
Generate blog topics using these templates:
- "How to [achieve outcome] without [common pain point]"
- "[Number] [positive adjective] ways to [achieve goal]"
- "[Product] vs [Competitor]: [Year] Comparison"
- "The Complete Guide to [topic] for [audience]"
- "Why [common approach] Doesn't Work (And What to Do Instead)"
- "[Industry] Trends in [Year]: What [audience] Should Know"
- "I [did something] for [time period]. Here's What Happened."

### 7. Local SEO (if applicable)
- Google Business Profile optimized
- NAP consistency across directories
- Location-specific landing pages
- Local keywords in title tags and content
- Schema markup for local business

## Scoring System
Rate each page 0-100:
- **Content Quality**: 0-30 points
- **On-Page Optimization**: 0-25 points
- **Technical Health**: 0-20 points
- **Link Profile**: 0-15 points
- **User Experience**: 0-10 points

## Output Format
When auditing a site, structure findings as:
1. **Critical SEO Issues** (blocking ranking, fix NOW)
2. **High-Impact Keyword Opportunities** (with search volume estimates)
3. **Content Strategy Recommendations** (specific blog post titles to write)
4. **Technical Fixes** (with priority ranking)

Each finding should include:
- The specific issue/opportunity
- WHY it matters for rankings
- Exact fix or implementation steps
