# Page CRO (Conversion Rate Optimization) Skill

## Frameworks

### 1. The LIFT Model (by WiderFunnel)
Evaluate every landing page element through these 6 factors:
- **Value Proposition**: Is the unique value immediately clear within 5 seconds?
- **Relevance**: Does the page match visitor intent / ad promise?
- **Clarity**: Is the message unambiguous? Can a 12-year-old understand it?
- **Urgency**: Is there a reason to act NOW vs. later?
- **Anxiety**: What fears/doubts does the visitor have? Are they addressed?
- **Distraction**: What elements pull attention AWAY from the primary CTA?

### 2. The AIDA Landing Page Audit
Check each stage:
- **Attention**: Does the hero grab attention in <3 seconds? (contrast, motion, bold claim)
- **Interest**: Does the subhead create curiosity or identify a pain point?
- **Desire**: Does the body build desire with benefits > features, social proof, outcomes?
- **Action**: Is the CTA clear, singular, and frictionless?

### 3. Trust Signal Checklist
Score 0-10 on these trust signals:
- [ ] Logo bar / "As seen in" section
- [ ] Customer testimonials with real names + photos
- [ ] Case study numbers (quantified results)
- [ ] Security badges / certifications
- [ ] Money-back guarantee or free trial
- [ ] Customer count or usage stats
- [ ] Real team photos / founder story
- [ ] Live chat or visible support

### 4. CTA Optimization Framework
Evaluate every CTA against:
- **Specificity**: "Start My Free Trial" > "Sign Up" > "Submit"
- **Value Echo**: CTA should echo the value prop, not describe the action
- **Visual Hierarchy**: Is the primary CTA the most visually dominant element?
- **Friction Score**: How many fields / steps before the user gets value?
- **Placement**: Above fold + repeated after each content section?

### 5. Above-the-Fold Audit (5-Second Test)
In 5 seconds, can a visitor answer:
1. What does this product/service do?
2. Who is it for?
3. Why should I care?
4. What do I do next?

### 6. Mobile CRO Checklist
- [ ] Thumb-friendly CTA buttons (min 48px tap target)
- [ ] No horizontal scrolling
- [ ] Forms use appropriate mobile keyboard types
- [ ] Images are optimized (not blocking content)
- [ ] Sticky CTA on scroll
- [ ] Phone number is click-to-call
- [ ] Critical content visible without scrolling

### 7. Page Speed Impact Rules
- Every 1-second delay = ~7% drop in conversions
- Page should load in <3 seconds on 3G
- Largest Contentful Paint (LCP) < 2.5s
- Cumulative Layout Shift (CLS) < 0.1

## Output Format
When auditing a page, structure findings as:
1. **Critical Conversion Killers** (fix immediately)
2. **High-Impact Opportunities** (fix this week)
3. **Quick Wins** (fix today)

Each finding should include:
- The specific element/text that's problematic
- WHY it hurts conversion (reference framework)
- A concrete rewrite or fix suggestion
